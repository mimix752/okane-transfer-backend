QUESTION: Comment retirer un transfert ?
ANSWER: Rendez-vous dans une agence Okane Transfer avec votre pièce d'identité et le code de retrait. L'agent vérifie le code et votre identité, puis procède au paiement.

QUESTION: retirer de l'argent
ANSWER: Présentez-vous chez un agent avec le code de retrait à 8 caractères et une pièce d'identité valide. L'agent vous remettra les fonds après vérification.

QUESTION: comment encaisser un transfert
ANSWER: Apportez le code de retrait et votre CIN ou passeport à une agence. L'agent validera les informations et vous remettra l'argent.

QUESTION: Quels documents sont nécessaires pour le retrait ?
ANSWER: Une pièce d'identité valide (CIN ou passeport) et le code de retrait communiqué par l'expéditeur sont obligatoires pour tout retrait.

QUESTION: Puis-je retirer sans pièce d'identité ?
ANSWER: Non, la présentation d'une pièce d'identité est obligatoire pour des raisons légales et de conformité. Aucun retrait ne peut être effectué sans vérification d'identité.

QUESTION: Que faire si j'ai perdu mon code de retrait ?
ANSWER: Contactez l'expéditeur pour qu'il vous communique à nouveau le code. L'expéditeur peut aussi le retrouver dans son historique de transferts.

QUESTION: je n'ai pas le code de retrait
ANSWER: Demandez à l'expéditeur de vous transmettre le code à 8 caractères reçu lors de l'envoi. Il figure aussi sur le reçu du transfert.

QUESTION: Le code de retrait est introuvable.
ANSWER: Vérifiez les caractères saisis, le code est sensible à la casse. Assurez-vous que le transfert est bien en statut EN_ATTENTE et non EXPIRÉ ou ANNULÉ.

QUESTION: Le code de retrait ne fonctionne pas.
ANSWER: Vérifiez que vous saisissez tous les caractères correctement. Si le problème persiste, contactez l'agence — le transfert peut être expiré ou annulé.

QUESTION: Pourquoi mon retrait est refusé ?
ANSWER: Le retrait peut être refusé si le transfert est expiré, annulé, déjà payé, ou si la pièce d'identité présentée ne correspond pas aux informations enregistrées.

QUESTION: retrait refusé
ANSWER: Vérifiez le statut du transfert et la concordance entre votre identité et les informations du bénéficiaire. Contactez l'agent pour plus de détails.

QUESTION: Puis-je retirer dans une autre agence ?
ANSWER: Oui, vous pouvez retirer dans n'importe quelle agence active dans votre pays de destination, pas nécessairement celle désignée à l'envoi.

QUESTION: Puis-je envoyer quelqu'un d'autre retirer à ma place ?
ANSWER: Non, le retrait est strictement personnel. Le bénéficiaire désigné doit se présenter en personne avec sa propre pièce d'identité.

QUESTION: Le bénéficiaire est mineur, peut-il retirer ?
ANSWER: Des règles spécifiques s'appliquent pour les mineurs. Contactez l'agence concernée pour connaître les conditions applicables dans votre pays.

QUESTION: Quel est le délai pour retirer ?
ANSWER: Il n'y a pas de délai minimum, mais le transfert expire si le retrait n'est pas effectué dans les délais définis. Vérifiez la date d'expiration sur le reçu.

QUESTION: Jusqu'à quand puis-je retirer ?
ANSWER: Chaque transfert a une date d'expiration. Passé ce délai, le statut passe à EXPIRÉ et le retrait n'est plus possible. Consultez le reçu pour la date limite.

QUESTION: Combien de temps dure le retrait en agence ?
ANSWER: Le processus de retrait en agence prend généralement quelques minutes, le temps de vérifier le code et l'identité du bénéficiaire.

QUESTION: Le transfert est marqué PAYÉ mais je n'ai pas reçu l'argent.
ANSWER: Si le statut est PAYÉ mais vous n'avez pas reçu les fonds, contactez immédiatement l'agence où le retrait a été enregistré et signalez l'incident.

QUESTION: Puis-je retirer en espèces ?
ANSWER: Oui, les retraits en agence se font généralement en espèces dans la devise locale du pays de destination.

QUESTION: Le transfert est disponible dans quelle devise à la réception ?
ANSWER: Le bénéficiaire reçoit les fonds dans la devise locale de l'agence de paiement, selon le taux de change appliqué au moment de l'envoi.

QUESTION: Comment obtenir un reçu de retrait ?
ANSWER: L'agent en agence imprime automatiquement un reçu de paiement après chaque retrait. Conservez-le comme preuve de réception.

QUESTION: Mon code a expiré avant que je puisse retirer.
ANSWER: Contactez l'expéditeur pour qu'il se rapproche de son agence et demande une prolongation ou un remboursement selon la politique en vigueur.

QUESTION: Puis-je retirer partiellement un transfert ?
ANSWER: Non, les transferts doivent être retirés en totalité. Le retrait partiel n'est pas possible.

QUESTION: retrait mobile money
ANSWER: Si votre transfert est destiné à un compte Mobile Money (Orange Money, Wave, M-Pesa), les fonds sont crédités directement sur le compte désigné sans passer par une agence.
