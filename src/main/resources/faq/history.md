QUESTION: Comment voir mon historique de transferts ?
ANSWER: Connectez-vous à votre espace client, puis accédez à la section "Historique". Tous vos transferts passés y sont listés avec leurs statuts et détails.

QUESTION: Où sont mes anciens transferts ?
ANSWER: Dans votre espace client, la section Historique affiche l'ensemble de vos transferts, des plus récents aux plus anciens.

QUESTION: je retrouve pas mon transfert
ANSWER: Connectez-vous et consultez votre historique. Utilisez les filtres par date, montant ou statut, ou recherchez par numéro de référence.

QUESTION: Comment filtrer mes transferts ?
ANSWER: Dans l'historique, vous pouvez filtrer par date, montant, statut (EN_ATTENTE, PAYÉ, ANNULÉ, EXPIRÉ) ou par corridor de destination.

QUESTION: Comment rechercher un transfert par référence ?
ANSWER: Dans la section Historique ou Suivi, entrez le numéro de référence du transfert dans le champ de recherche pour accéder directement à ses détails.

QUESTION: Puis-je exporter mon historique ?
ANSWER: Selon les fonctionnalités disponibles sur votre compte, un export de l'historique peut être possible. Consultez les options dans la section Historique.

QUESTION: Pourquoi un transfert n'apparaît plus dans l'historique ?
ANSWER: Tous les transferts restent dans l'historique. Si vous ne voyez pas un transfert, vérifiez vos filtres de date ou de statut — ils peuvent masquer certaines entrées.

QUESTION: Comment voir les détails d'un transfert ?
ANSWER: Cliquez sur un transfert dans la liste de l'historique pour afficher tous ses détails : montant, frais, bénéficiaire, statut, dates et code de retrait.

QUESTION: Combien de temps l'historique est-il conservé ?
ANSWER: L'historique de vos transferts est conservé de façon permanente dans votre espace client, conformément aux obligations légales de traçabilité.

QUESTION: Mon historique est vide.
ANSWER: Si vous êtes nouveau client, aucun transfert n'apparaît encore. Si vous avez déjà effectué des transferts, vérifiez que vous êtes connecté au bon compte.

QUESTION: Comment voir le statut d'un transfert spécifique ?
ANSWER: Dans l'Historique, cliquez sur le transfert concerné ou utilisez la section Suivi avec le numéro de référence pour voir le statut en temps réel.

QUESTION: Puis-je voir les transferts reçus ?
ANSWER: L'historique affiche les transferts effectués depuis votre compte. Pour les transferts reçus en tant que bénéficiaire, renseignez-vous auprès de l'agence de paiement.

QUESTION: Comment retrouver un ancien code de retrait ?
ANSWER: Dans l'historique, cliquez sur le transfert concerné. Le code de retrait est visible dans les détails si le transfert est encore actif.

QUESTION: Les transferts annulés apparaissent-ils dans l'historique ?
ANSWER: Oui, tous les transferts (EN_ATTENTE, PAYÉ, ANNULÉ, EXPIRÉ) apparaissent dans l'historique avec leur statut respectif.
