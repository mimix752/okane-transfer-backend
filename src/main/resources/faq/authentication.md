QUESTION: Comment créer un compte ?
ANSWER: Cliquez sur "S'inscrire", renseignez votre email, mot de passe et numéro de téléphone, puis validez le code OTP reçu par SMS.

QUESTION: Je veux m'inscrire.
ANSWER: Rendez-vous sur la page d'inscription, entrez vos informations personnelles et confirmez votre numéro de téléphone par SMS.

QUESTION: Comment ouvrir un compte Okane Transfer ?
ANSWER: L'inscription est disponible en ligne. Fournissez votre email, un mot de passe sécurisé et un numéro de téléphone valide pour recevoir le code de vérification.

QUESTION: Comment se connecter ?
ANSWER: Entrez votre email et mot de passe sur la page de connexion. Un code OTP peut être requis selon le niveau de sécurité de votre compte.

QUESTION: Je n'arrive pas à me connecter.
ANSWER: Vérifiez votre email et mot de passe. Si vous avez oublié votre mot de passe, utilisez l'option "Mot de passe oublié". Après 5 tentatives échouées, le compte est temporairement bloqué.

QUESTION: login ne fonctionne pas
ANSWER: Vérifiez que votre email est correctement saisi et que votre mot de passe est exact. En cas de blocage, attendez 10 minutes ou réinitialisez votre mot de passe.

QUESTION: J'ai oublié mon mot de passe.
ANSWER: Cliquez sur "Mot de passe oublié" sur la page de connexion. Vous recevrez un email avec un lien de réinitialisation valable 1 heure.

QUESTION: réinitialiser mot de passe
ANSWER: Depuis la page de connexion, cliquez sur "Mot de passe oublié" et suivez les instructions envoyées par email.

QUESTION: Comment changer mon mot de passe ?
ANSWER: Allez dans Paramètres > Sécurité > Changer le mot de passe. Saisissez votre mot de passe actuel puis le nouveau.

QUESTION: changer password
ANSWER: Rendez-vous dans vos paramètres de compte, section Sécurité, pour modifier votre mot de passe.

QUESTION: Mon compte est bloqué.
ANSWER: Après 5 tentatives de connexion échouées, le compte est bloqué pendant 10 minutes. Si le problème persiste, contactez le support.

QUESTION: compte suspendu
ANSWER: Un compte peut être suspendu suite à une activité suspecte ou une vérification KYC incomplète. Contactez un agent pour plus d'informations.

QUESTION: Qu'est-ce que la double authentification ?
ANSWER: La 2FA envoie un code OTP par SMS à chaque connexion ou opération sensible, ajoutant une couche de sécurité supplémentaire à votre compte.

QUESTION: À quoi sert le code OTP ?
ANSWER: Le code OTP (One-Time Password) est un code unique envoyé par SMS pour vérifier votre identité lors de la connexion ou d'une opération sensible.

QUESTION: Je ne reçois pas le code SMS.
ANSWER: Vérifiez que votre numéro de téléphone est correct dans votre profil. Attendez quelques secondes et demandez un nouveau code si nécessaire.

QUESTION: Le code OTP est expiré.
ANSWER: Les codes OTP sont valides 5 minutes. Si le vôtre a expiré, demandez un nouveau code via le bouton "Renvoyer le code".

QUESTION: J'ai reçu plusieurs codes OTP.
ANSWER: Seul le dernier code reçu est valide. Les codes précédents sont automatiquement invalidés.

QUESTION: Comment se déconnecter ?
ANSWER: Cliquez sur votre avatar en haut à droite puis sur "Se déconnecter". Votre session est immédiatement fermée.

QUESTION: ma session a expiré
ANSWER: Pour des raisons de sécurité, les sessions expirent après une période d'inactivité. Reconnectez-vous pour continuer.
